# Led Wall desk

---

## Description

Intégration à l'Environnement HomeKit apple de led qui décore et éclaire mon miroir.

---

## Matériels

- ESP-01S (esp8266)
- LEDs WS2815 (12V)
- Alimentation (12V)
- PLA (printing filament)
