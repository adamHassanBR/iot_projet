# capteur-qualit-de-l-air

-------------------------------------
## Description ##
  
  Integration à l'environement HomeKit apple de mon capteur de qualité de l'air (température, humidité et CO2)
  

-------------------------------------
## Matériels ##

* Wemos d1 mini (esp8266)
* DHT22
* Capteur MQ-2
* PLA (printing filament)
