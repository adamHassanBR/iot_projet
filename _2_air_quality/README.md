# Environment sensor

-------------------------------------
## Description ##
  
  Integration à l'environement HomeKit apple de mon capteur de qualité de l'air (température, humidité)
  

-------------------------------------
## Matériels ##

* ESP8266-01S
* DHT22
* PLA (printing filament)
