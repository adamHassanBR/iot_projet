# Environment sensor

-------------------------------------
## Description ##
  
Intégration à l'environnement HomeKit apple de mon capteur de qualité de l'air (température, humidité)
  

-------------------------------------
## Matériels ##

* ESP8266-01S
* DHT22
* PLA (printing filament)
