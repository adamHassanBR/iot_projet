# lego-city

-------------------------------------
## Description ##
  
  Intégration à l'environnement HomeKit apple de mes constructions lego city modifié avec des leds.

  
  Youtube link : https://www.youtube.com/watch?v=nuTSD4KbFxw

-------------------------------------
## Matériels ##

* Wemos d1 mini (esp8266)
* Leds blanc
* lego city Paris
* lego city NYC
* PLA (printing filament)
